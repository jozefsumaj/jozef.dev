# Blog Tags Reference

Use lowercase, hyphenated tags. Pick 1-3 relevant tags per post.

## Core SRE
| Tag | Use for |
|-----|---------|
| `reliability` | System reliability, resilience, chaos engineering |
| `incidents` | Incident management, on-call, response |
| `postmortems` | Learning from failure, blameless culture |

## Observability
| Tag | Use for |
|-----|---------|
| `observability` | General observability topics |
| `metrics` | Prometheus, Datadog, time-series data |
| `logging` | Log aggregation, structured logs, ELK |
| `tracing` | Distributed tracing, Jaeger, OpenTelemetry |
| `alerting` | Alert design, reducing noise, PagerDuty |

## Deployment & Releases
